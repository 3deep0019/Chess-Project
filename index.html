Instructions:- How to move queen from it's initial position

In the input box the text will be (S3)

S means South

3 means number of steps

Number should from 1 to 7 other than this number you will get an Error

  • N = North
  • S = South
  • E = East
  • W = West
  • NE = NorthEast
  • NW = NotherWest
  • SE = SouthEast
  • SW = SouthWest
